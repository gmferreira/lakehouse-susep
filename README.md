## lakehouse-susep
[pt-br] Projeto de Lakehouse no Databricks com foco nos dados públicos do SES/SUSEP, organizando camadas Bronze, Silver e Gold para ingestão, tratamento e análise de dados regulatórios do mercado de seguros.

[en-us] Lakehouse project at Databricks focusing on public data from SES/SUSEP, organizing Bronze, Silver, and Gold layers for ingestion, processing, and analysis of regulatory data from the insurance market.
